This Deep_Shallow_Interactions repository provides the python codes used to systematically and statistically identify interactions between intermediate-depth and shallow seismicity in the Tohoku area, as well as to characterize their duration and associated spatial pattern.

In Synchronisation_seismicity_rates, we use correlation to statistically look for synchronized day-long bursts of intermediate depth and shallow seismicity.
In Search_lineaments_clustering, we use clustering methods to find and study short duration, along-dip earthquake lineaments.
